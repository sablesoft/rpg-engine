# Faction Emblem Template

## Best For

- heraldic marks
- seals
- banner devices
- icon systems for factions or cults

## Prompt Structure

Faction Identity:
- faction name
- ideology
- reputation

Symbol Logic:
- celestial symbols
- shapes
- ritual geometry
- culturally meaningful motifs

Graphic Style:
- carved, painted, stitched, embossed, inked, mosaic, metalwork

Presentation:
- isolated emblem
- banner fragment
- wax seal

## Negative Prompt Focus

- clip-art simplicity unless explicitly requested
- modern logos or corporate branding language
- symbols that contradict the faction's worldview
- unreadable symmetry, accidental text, extra decorative clutter
