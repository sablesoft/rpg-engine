# Location Image Template

## Best For

- establishing shots
- location concept art
- travel destination plates

## Prompt Structure

Place Identity:
- name and function
- world context
- sacred, political, or practical role

Spatial Read:
- foreground, midground, background
- scale cues
- access points, pathways, thresholds, verticality

Atmosphere:
- weather
- ritual activity
- sound implied through visual detail
- celestial phenomena or omen traces

Material Language:
- stone, ash, nacre, brass, vellum, bone, glass, woven banners, sea-light

## Negative Prompt Focus

- modern city elements
- industrial machinery
- overcrowded noise when the location mood is ceremonial or solemn
- warped architecture, malformed perspective, unreadable focal structure
