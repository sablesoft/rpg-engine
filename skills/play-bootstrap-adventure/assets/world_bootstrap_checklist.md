# Play Bootstrap Checklist

Before entering live play, ensure all of the following are established:

- one selected or newly created `world`
- one selected or newly created `scenario` tied to that world
- one selected or newly created protagonist `character` tied to that world
- one `adventure` instance built from those inputs
- an opening scene with immediate tension
- enough known facts and goals to let the first turn start naturally

Do not enter live play with missing core inputs.
