# Scenario Workspace Structure

Root:
- `products/rpg-engine/workspaces/scenario/<slug>/scenario.md`

Optional support files:
- `products/rpg-engine/workspaces/scenario/<slug>/global_story.md`
- `products/rpg-engine/workspaces/scenario/<slug>/tone_and_themes.md`

Optional support-card folders:
- `products/rpg-engine/workspaces/scenario/<slug>/quests/`
- `products/rpg-engine/workspaces/scenario/<slug>/locations/`
- `products/rpg-engine/workspaces/scenario/<slug>/characters/`

Primary card:
- `scenario.md` is the main readable source of truth

When to use:
- use for global scenario canon promoted to workspace scope
- do not use for local scenario content that still belongs only to one adventure
